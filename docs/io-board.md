# Placa de Entrada/Saída

A placa de entrada e saída possui vários componentes para controle externo do robô e está ligada diretamente ao microcontrolador.

## Entradas

### Botões

| Nome | Função |
|:---:|:---|
| BUTTON0 | Reseta o robô para a condição inicial |
| BUTTON1 | Alterna entre o modo manual e automático de execução |
| BUTTON2 | Inicia a caminhada caso o modo seja manual |
| BUTTON3 | *A definir* |
| BUTTON4 | *A definir* |

## Saídas

### Leds

| Nome | Função |
|:---:|:---|
| LED0 | Ligado se o rosserial está conectado |
| LED1 | Ligado quando o robô está no modo manual |
| LED2 | *A definir* |
| LED3 | *A definir* |
| LED4 | Led para debug ao longo da execução (Inicialização, procura por motores e envio de mensagens para os motores) |
